# 用markdown来画流程图
## Example
代码  

``` 
flow  
st=>start: Start  
op=>operation: Your Operation  
cond=>condition: Yes or No?  
e=>end  
st->op->cond  
cond(yes)->e  
cond(no)->op  
```
